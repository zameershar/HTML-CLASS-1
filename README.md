## 1st Class Outline
1. Headings `<H1>`, `<H2>` .... `<H6>`
2. Bold tag = `<b>`
3. Italic = `<i>`
4. Underline = `<u>`
5. Center = `<center>`
6. Break = `<br>`
7. Heading Row = `<hr>`

## 2nd Class Outline
1. Order List = `<ol>`
2. Unorder List = `<ul>`
3. Description List = `<dl>`
4. Tables = `table>`
